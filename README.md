# 代理配置教程

请先看你正在使用的设备，然后点开对应教程。其它设备的教程不用看。

## 选择你的设备

### iPhone / iPad

打开：[iOS 配置教程](docs/ios.md)

开始前需要拿到：服务器地址和密码。

### Mac 电脑

打开：[macOS 配置教程](docs/macos.md)

开始前需要拿到：`personal.yaml` 文件。

### Windows 电脑

打开：[Windows 配置教程](docs/windows.md)

开始前需要拿到：`personal.yaml` 文件。

## 注意

服务器地址、密码、节点文件不要发到群里，也不要发给无关人员。
